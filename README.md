# Mosaic Kitchen — static starter

This repository contains a static HTML/CSS/JS starter for "Mosaic Kitchen" — a portfolio-ready restaurant site (African + international fusion).

Files:
- index.html — main page
- styles.css — styles and branding
- script.js — minimal client-side handler for the reservation form

How to run locally:
1. Clone the repo
2. Serve the folder with a static server (recommended) or open index.html in your browser
   - Python 3: `python -m http.server 5173`
   - Node: `npx serve .`

Notes:
- Images use Unsplash placeholder URLs; replace with your own optimized images before publishing.
- This repo is public and GitHub Pages is enabled (publish from main branch) per your request.
